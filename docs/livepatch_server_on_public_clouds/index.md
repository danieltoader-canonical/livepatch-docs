# Livepatch Server On Public Clouds

Livepatch server can be self-hosted on public clouds via the server snap, allowing users to leverage public cloud resources while still having fine grain control over patch roll-out on their machines.

Canonical customers can receive support and report issues on the Ubuntu Livepatch service, Livepatch client or Livepatch on-prem, on Canonical's [support portal](https://portal.support.canonical.com/).
