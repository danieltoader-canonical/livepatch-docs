# Supported architectures

Canonical currently provides live kernel patches for kernels on the following CPU architectures:

- **amd64** (x86-64)
- **s390x**

Support for ARM64 (arm64) is available starting with Ubuntu 26.04 LTS. Older Ubuntu releases are not covered for ARM64 due to limitations in the tooling available in those releases.
